Collatz-Problem

1.1 Ablauf

  • Wenn n ungerade → n nimmt n/2 an.
  • Wenn n gerade → n nimmt 3n + 1 an.
  • Vorgang wiederholen

1.2 Beispiele

8 → 4 → 2 → 1 → 4 → ...

5 → 16 → 8 → 4 → 2 → 1 → 4 → ...

1.3 Folgerung

Jede m ogliche Zahlenfolge einer Startzahl n ∈ N \ {0} endet im Folgeschema 4 → 2 → 1 → ...

1.4 Problem

1.3 ist nicht bewiesen.

1.5 Collatz-Schritte

Als Länge der Collatz-Folge wird die Anzahl der Schritte bezeichnet, die benötigt wird, um zum Wert 1 zu gelangen.
